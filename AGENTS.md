## Project Summary
A high-fidelity clone of the Xcelens website (xcelens.club), a personal coaching platform. The project features a sophisticated "Corporate Serene" aesthetic with cinematic imagery, smooth scroll animations, and a structured grid-based layout.

## Tech Stack
- Framework: Next.js 15 (App Router)
- Styling: Tailwind CSS 4, Framer Motion
- Icons: Lucide React
- Components: Radix UI (Shadcn UI compatible)
- Animation: tw-animate-css, framer-motion

## Architecture
- `src/components/sections/`: Contains individual website sections (Navbar, Hero, About, etc.)
- `src/app/page.tsx`: Main entry point integrating all sections
- `src/app/globals.css`: Global styles, themes, and custom Tailwind v4 utilities

## User Preferences
- Pixel-perfect cloning of animations, images, and scroll effects
- High visual accuracy and distinctive typography

## Project Guidelines
- Use Tailwind v4 `@utility` for custom utility classes
- Maintain a clean, minimalist code structure without unnecessary comments
- Follow the established vertical grid line layout system

## Common Patterns
- Floating pill-shaped navigation
- Section tags for semantic grouping
- Cinematic hero and CTA sections with parallax-like depth
